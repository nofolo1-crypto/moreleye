🍄 Morel Field Guide

AR Overlay Guide

  • Orange box + label = host tree detected on screen
  • Blue arrow on ground = walk toward that tree
  • Green dashed circle = prime search zone around tree
  • Green glowing outline = morel mushroom detected

Morel ID

  • Honeycomb / pitted cap — not smooth
  • Fully hollow when sliced vertically
  • Cap fused to stem at base
  • Tan, grey, yellow-brown, or black
  • 2–6 inches tall; season March–May

Host Trees

  • Dead / dying elm — best single indicator
  • Ash killed by Emerald Ash Borer
  • Old apple and fruit trees
  • Tulip poplar, sycamore, cottonwood
  • Mature oaks — later in season

⚠ False Morel Warning

  • Gyromitra — brain-like cap, NOT pitted
  • Cap NOT fused to stem
  • Not fully hollow inside
  • TOXIC — always do the hollow slice test
